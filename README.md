# 🐍 Python: Crie a sua primeira aplicação

Este repositório foi criado para armazenar o progresso, exercícios e projetos práticos realizados durante o curso de Fundamentos de Python. O foco principal é aprender a sintaxe da linguagem, estruturas de dados e boas práticas de desenvolvimento.



## 📂 Estrutura do Repositório

O projeto está organizado por módulos, seguindo a cronologia do aprendizado:

* **01-manipulacao-de-strings**: Primeiros passos com `print`, entrada de dados com `input` e formatação de textos (f-strings).
* **02-modulos-e-funcoes**: Organização de código com `def` e uso de módulos padrão do Python como `os`.
* **03-lista-lacos-e-excecoes**: Trabalho com coleções (listas), repetições (`for` e `while`) e controle de erros com `try-except`.
* **04-dicionarios**: Armazenamento de dados complexos utilizando o conceito de chave e valor.
* **05-consolidando-conhecimentos**: Projeto final prático aplicando todos os conceitos anteriores em uma aplicação CLI.

## 🛠️ Tecnologias Utilizadas
